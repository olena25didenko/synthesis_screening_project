# Synthesis Screening Robustness Project

## Overview
This project evaluates DNA synthesis screening robustness against AI-assisted sequence evasion.

**Note:** This is defensive research conducted under ERA AIxBio Fellowship supervision.

## Setup Instructions
See SETUP.md

## Reference Sequences
- Influenza A HA
- SARS-CoV-2 Spike
- Measles F protein
- Lassa virus NP
- Ebola VP40

All sequences are published, publicly available references.
